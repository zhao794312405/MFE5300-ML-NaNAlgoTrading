Make sure that you install all the packages in the requirements.txt


Firstly, run the Code_Fators.py to obtain the factor data of stock in hs300 form 2010-01-01 to 2020-03-31,saved as dict_factors.pickle 

Secondly, run the Strategy_turn_para.py to obtain the best random forest strategy.

Thirdly, run the Random_forest_strategy to obtain the selected stocks list saved as stock_list.pkl

Fourthly, run the Backtest.py to test the performance of our strategy.

(hs300.xlsx is the market data of hs300 index)
